# 🛒 eSales Automation Testing Project

## 📌 Project Overview
This repository contains an automated testing framework for the eSales application. 
The project demonstrates practical automation techniques for web application testing 
using industry‑standard tools and frameworks to ensure application reliability, stability, and accuracy.

## 🎯 Objectives
- Build a reusable and maintainable automation framework
- Automate end‑to‑end test cases for critical workflows
- Validate UI elements and business logic functionality
- Improve test efficiency and coverage through automation

## 🛠️ Tools & Technologies
- Selenium WebDriver  
- Java 
- TestNG for test management  
- Maven for dependency management  
- Page Object Model (POM) design pattern  
- Git & GitHub for version control

## 📋 Test Coverage
- Functional test automation  
- Regression test automation  
- UI automation validation  
- Form input and navigation testing  

## 🔄 Key Automated Scenarios
- User login and authentication  
- Product selection and browsing  
- Cart operations (add/remove)  
- Checkout process validation  
- Navigation and UI element checks  

## 🧪 Framework Design
- **Page Object Model (POM)** for better maintainability and readability  
- Reusable methods and utility helpers  
- Structured test suite with TestNG  
- Reporting and logging for test execution results

## 🐞 Defect Management
- Captured application defects during automation runs  
- Logged issues with steps, screenshots, and expected vs actual results  
- Re‑executed automated tests after defect fixes to ensure stability

## 🔁 Automation Workflow
1. Identify test cases for automation  
2. Setup automation framework structure  
3. Develop script using Selenium WebDriver  
4. Execute tests via TestNG  
5. Generate reports and logs  
6. Review results and update scripts as needed

## 📊 Outcome
- Enhanced reliability and consistency of test execution  
- Reduced manual testing time  
- Improved confidence in release quality  
- Robust automation foundation for future enhancements
